# 视频批量下载 & 链接提取工具集

三个工具合一：Web 端批量下载 + B站链接提取 + 抖音链接提取。双击即用。

## 项目结构

```

├── main.py                     ← Web 服务入口
│
├── 启动批量/                    ← 批量下载前端页面
│   ├── index.html              ← Web UI（单条/批量下载）
│   ├── dy.js                   ← 业务逻辑
│   ├── main.cmd                ← 快捷启动
│   └── node.exe                ← 运行时
│
├── 获取链接/
│   ├── B站获取链接/              ← B站视频链接提取
│   │   ├── 启动.bat
│   │   ├── get_bilibili_links.py
│   │   └── README.md
│   └── 抖音获取链接/             ← 抖音视频链接提取
│       ├── 启动.bat
│       ├── get_favorites.py
│       └── README.md
│
├── parser/                     ← 各平台解析器（22个平台）

```

## 工具一：Web 视频下载器

### 启动
双击 `main.bat`，自动打开浏览器进入下载页面。

### 单条下载
粘贴视频分享链接（抖音/B站/小红书/快手等 22 个平台），点击下载。

### 批量下载
每行一个链接，支持从 B站/抖音提取工具输出的 `视频链接.txt` 直接粘贴。

## 工具二：B站链接提取

双击 `获取链接\B站获取链接\启动.bat`

1. 自动打开浏览器 → B站个人空间
2. 登录 → 进入目标页面（收藏夹/UP主列表/搜索）
3. 终端按 Enter 开始提取
4. 链接保存到 `bilibili_视频链接.txt`

## 工具三：抖音链接提取

双击 `获取链接\抖音获取链接\启动.bat`

1. 自动打开浏览器 → 抖音首页
2. 登录 → 进入收藏页面
3. 终端按 Enter 开始提取
4. 链接保存到 `douyin_视频链接.txt`

## 常见问题

**Q: 浏览器没反应？** 系统需装有 Chrome 或 Edge，程序自动选择可用浏览器。

**Q: 提取到无关链接？** 已内置可见性过滤，只提取页面上真实可见的视频卡片。

**Q: 下载失败？** 单条链接超 3 分钟自动跳过，失败链接保存到 `failed_logs` 文件夹，可用 `main.bat` 二次尝试。
